<summary><b>테마 정보 (Theme Information)</b></summary>

이 블로그는 [sighingnow/jekyll-gitbook](https://github.com/sighingnow/jekyll-gitbook) 테마를 기반으로 제작되었습니다. 

The search functionality in jekyll-gitbook theme is powered by the [gitbook-plugin-search-pro][5] plugin and is enabled by default.

The code highlight style is configurable the following entry in `_config.yaml`:

```yaml
syntax_highlighter_style: colorful
```

For more information on the theme, please visit the [original repository](https://github.com/sighingnow/jekyll-gitbook).
