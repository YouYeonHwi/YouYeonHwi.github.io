파이썬에서는 조건에 따라 코드 실행 흐름을 제어하거나 반복적으로 작업을 수행할 수 있습니다. 여기서는 프로그램의 기본 구조를 이루는 조건문과 반복문을 알아봅니다.

## 조건문

조건문은 특정 조건이 참(True)인지 거짓(False)인지에 따라 서로 다른 코드를 실행합니다. 대표적인 문법은 `if`, `elif`, `else`입니다.

```python
age = 18

if age >= 20:
    print("성인입니다.")
elif 13 <= age < 20:
    print("청소년입니다.")
else:
    print("어린이입니다.")
```

* `if` 다음에는 조건식을 적고, 조건이 참이면 블록 안의 코드를 실행합니다.
* `elif`는 이전 조건이 거짓일 때 추가로 조건을 검사합니다.
* `else`는 위의 모든 조건이 거짓일 때 실행됩니다.

## 반복문

반복문은 일정한 동작을 여러 번 반복할 때 사용합니다. 파이썬에서는 주로 `for`와 `while`을 사용합니다.

### `for` 반복문

`for` 문은 시퀀스(리스트, 문자열 등)나 반복 가능한(iterable) 객체의 각 요소에 대해 코드를 실행합니다.

```python
fruits = ["사과", "배", "귤"]
for fruit in fruits:
    print(fruit)
```

`range()` 함수를 이용하면 정해진 횟수만큼 반복할 수 있습니다.

```python
for i in range(3):
    print(i)
# 출력: 0 1 2
```

### `while` 반복문

`while` 문은 조건이 참인 동안 반복합니다. 조건이 계속 참이면 무한 루프가 되므로 주의해야 합니다.

```python
count = 0
while count < 3:
    print(count)
    count += 1
# 출력: 0 1 2
```

### 반복 제어

반복문 안에서 `break`와 `continue`를 사용해 흐름을 제어할 수 있습니다.

```python
for i in range(5):
    if i == 3:
        break      # i가 3이면 반복 종료
    if i % 2 == 0:
        continue   # 짝수는 아래 코드를 건너뜀
    print(i)       # 출력: 1
```

* `break`는 반복문을 즉시 종료합니다.
* `continue`는 현재 반복을 건너뛰고 다음 반복으로 넘어갑니다.

조건문과 반복문을 적절히 사용하면 프로그램의 흐름을 세밀하게 제어할 수 있으며, 복잡한 작업도 간결하게 표현할 수 있습니다.
