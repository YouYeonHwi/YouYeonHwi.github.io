## 빠른 비교

| 자료형 | 변경 가능 여부 | 순서 보장 | 중복 허용 |
| --- | --- | --- | --- |
| 숫자형(`int`, `float`, `complex`) | 불변 | 해당 없음 | 해당 없음 |
| 문자열(`str`) | 불변 | O | O |
| 리스트(`list`) | 가변 | O | O |
| 튜플(`tuple`) | 불변 | O | O |
| 딕셔너리(`dict`) | 가변 | O (Python 3.7+) | 키 중복 X |
| 집합(`set`) | 가변 | X | X |
| 불 자료형(`bool`) | 불변 | 해당 없음 | 해당 없음 |
