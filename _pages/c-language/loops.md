## 1. 반복문이란?

반복문은 특정 코드 블록을 **조건이 만족되는 동안 반복**하여 실행하게 해주는 구문입니다. 동일한 작업을 여러 번 수행할 때 반복문을 사용하면 코드를 훨씬 간결하게 만들 수 있습니다.

C언어의 반복문에는 `for`, `while`, `do-while`이 있습니다.

## 2. `for` 문: 횟수가 정해진 반복

반복 횟수가 명확할 때 가장 많이 사용하는 반복문입니다.

**기본 구조:**
```c
for (초기식; 조건식; 증감식) {
    // 조건식이 참인 동안 반복할 코드
}
```
- **초기식**: 반복 시작 전 딱 한 번 실행 (보통 카운터 변수 초기화)
- **조건식**: 각 반복 전에 검사 (거짓이 되면 반복 종료)
- **증감식**: 각 반복이 끝난 후 실행 (보통 카운터 변수 변경)

**예제:**
```c
#include <stdio.h>

int main() {
    // 1부터 5까지 출력
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n"); // 출력: 1 2 3 4 5

    // 1부터 100까지의 합
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum += i;
    }
    printf("1~100의 합: %d\n", sum); // 5050

    return 0;
}
```

## 3. `while` 문: 조건 기반 반복

반복 횟수를 미리 알 수 없고, **조건이 참인 동안** 계속 반복할 때 사용합니다. 조건을 먼저 검사하므로 처음부터 조건이 거짓이면 한 번도 실행되지 않을 수 있습니다.

**기본 구조:**
```c
while (조건식) {
    // 조건식이 참인 동안 반복할 코드
    // 반드시 조건을 false로 만드는 코드가 있어야 함!
}
```

**예제:**
```c
#include <stdio.h>

int main() {
    // 1부터 5까지 출력
    int i = 1;
    while (i <= 5) {
        printf("%d ", i);
        i++; // 이 코드가 없으면 무한루프!
    }
    printf("\n"); // 출력: 1 2 3 4 5

    return 0;
}
```

> **무한 루프 주의**: `while` 문 안에서 조건을 `false`로 만드는 코드가 없으면 프로그램이 영원히 종료되지 않는 **무한 루프**에 빠집니다.

## 4. `do-while` 문: 최소 한 번은 실행

`while` 문과 유사하지만, **코드 블록을 먼저 실행한 후 조건을 검사**합니다. 따라서 조건이 처음부터 거짓이더라도 코드 블록은 최소 한 번 실행됩니다.

**기본 구조:**
```c
do {
    // 실행할 코드 (처음엔 조건 검사 없이 무조건 실행)
} while (조건식); // 세미콜론 필수!
```

**예제 (메뉴 입력):**
```c
#include <stdio.h>

int main() {
    int choice;

    do {
        printf("--- 메뉴 ---\n");
        printf("1. 게임 시작\n");
        printf("2. 설정\n");
        printf("0. 종료\n");
        printf("선택: ");
        scanf("%d", &choice);
        printf("선택한 항목: %d\n\n", choice);
    } while (choice != 0); // 0을 입력할 때까지 반복

    printf("프로그램을 종료합니다.\n");
    return 0;
}
```

## 5. `break`와 `continue`

반복문 내에서 흐름을 제어하는 데 사용합니다.

- **`break`**: 반복문을 즉시 탈출합니다.
- **`continue`**: 현재 반복의 나머지 코드를 건너뛰고 다음 반복으로 넘어갑니다.

```c
#include <stdio.h>

int main() {
    printf("--- break 예제 ---\n");
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break; // i가 5가 되면 반복문 탈출
        }
        printf("%d ", i);
    }
    printf("\n"); // 출력: 1 2 3 4

    printf("--- continue 예제 ---\n");
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            continue; // 짝수면 건너뜀
        }
        printf("%d ", i); // 홀수만 출력
    }
    printf("\n"); // 출력: 1 3 5 7 9

    return 0;
}
```

반복문은 C언어에서 가장 자주 사용하는 구문 중 하나입니다. `for`, `while`, `do-while`의 차이점을 이해하고, 상황에 맞는 반복문을 선택하는 것이 좋은 코드를 작성하는 핵심입니다.
