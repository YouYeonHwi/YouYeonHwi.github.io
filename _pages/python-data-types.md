파이썬에서 데이터를 표현하는 방법을 이해하면 프로그램을 더 쉽게 작성할 수 있습니다. 자료형(Data Type)은 값의 종류와 그 값에 사용할 수 있는 연산을 결정합니다. 여기서는 가장 기본이 되는 네 가지 자료형을 살펴보겠습니다.

## 숫자형 자료형

숫자형은 수치를 다루는 자료형입니다. 종류에 따라 다음과 같이 나뉩니다.

- **정수(int)**: 소수점이 없는 숫자입니다. `-3`, `0`, `42`와 같이 표현합니다.
- **실수(float)**: 소수점을 포함한 숫자입니다. `3.14`, `-0.001` 등이 있습니다.
- **복소수(complex)**: `a + bj` 형태로 표현되는 숫자로, 공학 계산 등에 사용됩니다.

숫자형끼리는 사칙연산을 비롯해 다양한 수학 연산을 수행할 수 있습니다. 서로 다른 숫자형끼리 연산을 하면 자동으로 형 변환이 이루어집니다.

```python
# 정수와 실수의 연산
x = 10      # int
y = 3.5     # float
result = x + y  # 결과는 float

# 복소수 연산
z = 1 + 2j
w = 2 - 1j

print(result)          # 13.5
print(z * w)            # (4+3j)
print(type(x), type(y), type(z))
```

## 문자열 자료형

문자열(`str`)은 텍스트를 표현하는 자료형입니다. 작은따옴표(`'...'`)나 큰따옴표(`"..."`)로 감싸서 만듭니다. 문자열은 순서가 있어 인덱싱과 슬라이싱이 가능하며, 한 번 생성하면 수정할 수 없는(immutable) 자료형입니다.

```python
greeting = "Hello, Python!"
print(greeting)       # Hello, Python!
print(greeting[0])    # H
print(greeting[7:13]) # Python

# 문자열 포맷팅
name = "Ada"
age = 30
print(f"{name}는 {age}살입니다.")
```

주요 메서드와 함수:
- `len(greeting)`: 문자열의 길이를 반환합니다.
- `greeting.lower()`: 모든 문자를 소문자로 변환합니다.
- `greeting.replace("Python", "World")`: 특정 단어를 다른 단어로 바꿉니다.

## 리스트 자료형

리스트(`list`)는 여러 값을 순서대로 저장할 수 있는 자료형으로, 값의 추가·수정·삭제가 가능합니다. 대괄호 `[]`를 사용하여 만들며, 서로 다른 자료형을 함께 저장할 수도 있습니다.

```python
numbers = [1, 2, 3]
numbers.append(4)      # 요소 추가
numbers[0] = 10        # 값 수정
del numbers[1]         # 요소 삭제
print(numbers)         # [10, 3, 4]
print(numbers[:2])     # [10, 3]
```

리스트의 특징:
- **순서가 있는** 자료형이므로 인덱스로 접근할 수 있습니다.
- **변경 가능**(mutable)하여 요소를 자유롭게 바꿀 수 있습니다.
- 중첩 리스트를 사용하면 2차원 이상의 구조도 만들 수 있습니다.

## 튜플 자료형

튜플(`tuple`)은 리스트와 비슷하지만, 한 번 생성하면 수정할 수 없는(immutable) 자료형입니다. 소괄호 `()`를 사용하며, 변경이 필요 없는 데이터를 안전하게 저장할 때 사용합니다.

```python
point = (2, 3)
# point[0] = 5  # TypeError 발생

# 언패킹
x, y = point
print(f"x={x}, y={y}")

# 여러 값을 한 번에 반환하는 함수에서 자주 사용

def divide(a, b):
    return a // b, a % b

quotient, remainder = divide(7, 3)
print(quotient, remainder)  # 2 1
```

튜플의 특징:
- **순서가 있는** 자료형 → 인덱스 사용 가능
- **변경 불가능**하여 고정된 데이터를 표현할 때 유리합니다.
- 리스트보다 메모리를 적게 사용합니다.
