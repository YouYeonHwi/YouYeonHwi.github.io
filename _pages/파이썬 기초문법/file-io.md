## 파일 입출력이란?

파일 입출력(File Input/Output)은 프로그램이 디스크에 저장된 **파일을 읽고 쓰는(read/write) 작업**을 의미합니다. 이를 통해 데이터를 영구적으로 저장하여 프로그램이 종료된 후에도 유지하거나, 대용량의 데이터를 처리하거나, 다른 프로그램과 데이터를 주고받을 수 있습니다.

파이썬에서는 `open()` 내장 함수를 사용하여 파일 객체를 얻고, 이 객체의 메서드를 통해 파일을 다룹니다.

## 파일 열기: `open()` 함수

`open()` 함수는 파일을 열고 파일 객체를 반환합니다.

**기본 구조:**
`open(file, mode='r', encoding=None)`

- **`file`**: 열고자 하는 파일의 경로 (예: `"my_data.txt"`, `"C:/Users/data.csv"`)
- **`mode`**: 파일을 여는 방식(모드)을 지정합니다. (아래 표 참고)
- **`encoding`**: 파일을 읽고 쓸 때 사용할 인코딩 방식을 지정합니다. 특히 한글을 다룰 때는 `'utf-8'`로 지정하는 것이 필수적입니다.

**파일 열기 모드 (mode)**

| 모드 | 설명                                                                                             | 파일이 없을 때     |
| :--- | :----------------------------------------------------------------------------------------------- | :----------------- |
| `r`  | **읽기 모드 (Read)**: 파일의 내용을 읽기만 할 때 사용합니다. (기본값)                            | 오류 발생          |
| `w`  | **쓰기 모드 (Write)**: 파일에 내용을 쓸 때 사용합니다. **기존 내용은 모두 삭제됩니다.**           | 새 파일 생성       |
| `a`  | **추가 모드 (Append)**: 파일의 **끝에** 새로운 내용을 추가할 때 사용합니다. 기존 내용을 유지합니다. | 새 파일 생성       |
| `x`  | **배타적 생성 모드 (Exclusive Creation)**: 파일이 없을 때만 쓰기 모드로 열 수 있습니다.           | 새 파일 생성       |
| `b`  | **바이너리 모드 (Binary)**: 텍스트가 아닌 이미지, 동영상 등 바이너리 파일을 다룰 때 `r`, `w`, `a`와 함께 사용됩니다. (예: `rb`, `wb`) | -                  |
| `t`  | **텍스트 모드 (Text)**: 텍스트 파일을 다룰 때 사용합니다. (기본값)                               | -                  |
| `+`  | **업데이트 모드 (Update)**: 읽기와 쓰기를 모두 할 수 있도록 `r`, `w`, `a`와 함께 사용됩니다. (예: `r+`, `w+`) | -                  |

## 파일 쓰기 (`w`, `a` 모드)

### `write()` 메서드

파일에 문자열을 씁니다. 줄바꿈이 자동으로 되지 않으므로, 필요하면 `
` 문자를 직접 추가해야 합니다.

```python
# 'w' 모드: 기존 내용을 덮어씀
f = open("new_file.txt", "w", encoding="utf-8")
f.write("안녕하세요, 파이썬 파일 쓰기 테스트입니다.")
f.write("두 번째 줄을 작성합니다.")
f.close() # 중요: 파일 작업이 끝나면 반드시 닫아야 합니다.

# 'a' 모드: 기존 내용 끝에 추가
f = open("new_file.txt", "a", encoding="utf-8")
f.write("이 내용은 파일 끝에 추가됩니다.")
f.close()
```

## `with` 문: 안전하게 파일 다루기 (꿀팁!)

파일을 열고 작업을 하다가 예외가 발생하거나, `close()`를 호출하는 것을 잊으면 파일이 제대로 닫히지 않아 데이터가 손상되거나 리소스가 낭비될 수 있습니다.

`with` 문을 사용하면, `with` 블록이 끝날 때 **파이썬이 자동으로 파일을 닫아주므로** 훨씬 안전하고 간결합니다.

**기본 구조:**
```python
with open(file, mode, encoding) as 파일_객체_변수:
    # 파일 작업 수행
# with 블록을 벗어나면 파일은 자동으로 닫힘
```

**예제:**
```python
# with 문을 사용한 파일 쓰기
with open("new_file_with.txt", "w", encoding="utf-8") as f:
    f.write("with 문을 사용하면 close()가 필요 없습니다.")
    f.write("자동으로 닫히므로 매우 편리합니다.")

# with 블록 바깥에서는 f 객체를 사용할 수 없음
# f.write("이 코드는 에러 발생")
```
> **강력 추천**: 특별한 이유가 없다면, 파일 입출력 시에는 항상 `with` 문을 사용하는 것이 좋습니다.

## 파일 읽기 (`r` 모드)

### `read()`: 파일 전체 내용 읽기

파일의 모든 내용을 하나의 문자열로 읽어옵니다. 파일이 매우 클 경우 메모리 문제를 일으킬 수 있으니 주의해야 합니다.

```python
with open("new_file_with.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
```

### `readline()`: 한 줄씩 읽기

파일의 내용을 한 번에 한 줄씩 읽어옵니다. `while` 문과 함께 사용하여 파일 끝까지 한 줄씩 처리할 때 유용합니다.

```python
print("--- readline()으로 읽기 ---")
with open("new_file_with.txt", "r", encoding="utf-8") as f:
    while True:
        line = f.readline()
        if not line:  # 더 이상 읽을 줄이 없으면(파일의 끝) 빈 문자열 반환
            break
        print(line.strip()) # strip()으로 양쪽의 공백 및 줄바꿈 문자 제거
```

### `readlines()`: 모든 줄을 리스트로 읽기

파일의 모든 줄을 각각의 문자열 요소로 갖는 리스트를 반환합니다. `for` 문과 함께 사용하기 편리합니다.

```python
print("--- readlines()로 읽기 ---")
with open("new_file_with.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    for line in lines:
        print(line.strip())
```

### `for` 문으로 직접 파일 객체 순회하기 (가장 파이썬스러운 방법)

파일 객체는 반복 가능한(iterable) 객체이므로, `for` 문을 사용해 바로 한 줄씩 읽어올 수 있습니다. 가장 간결하고 효율적인 방법입니다.

```python
print("--- for 문으로 직접 순회하기 ---")
with open("new_file_with.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```
> 대용량 파일을 처리할 때 `read()`는 메모리 문제를 일으킬 수 있지만, `readline()`이나 `for` 문을 사용한 순회는 한 번에 한 줄씩만 메모리에 올리므로 훨씬 효율적입니다.

파일 입출력은 데이터를 다루는 모든 프로그램의 기본입니다. 특히 `with` 문과 `for` 문을 활용한 파일 읽기 패턴은 파이썬에서 매우 자주 사용되므로 꼭 익혀두는 것이 좋습니다.
