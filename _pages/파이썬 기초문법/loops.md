## 반복문이란?

반복문은 **특정 작업을 여러 번 반복**해야 할 때 사용하는 프로그래밍 구조입니다. 반복문을 사용하면 코드를 여러 번 복사하여 붙여넣을 필요 없이, 간결하고 효율적으로 동일하거나 유사한 작업을 처리할 수 있습니다.

파이썬의 대표적인 반복문은 `for` 문과 `while` 문입니다.

## `for` 문: 정해진 횟수만큼 반복하기

`for` 문은 리스트, 튜플, 문자열과 같이 **순서가 있는 자료구조(시퀀스)나 반복 가능한 객체(iterable)의 각 요소를 하나씩 순회**하면서 코드 블록을 실행합니다. 반복 횟수가 명확할 때 주로 사용됩니다.

**기본 구조:**
```python
for 변수 in 반복_가능한_객체:
    # 각 요소에 대해 반복적으로 실행될 코드
```

**예제 1: 리스트 순회**
```python
fruits = ["사과", "바나나", "체리"]
for fruit in fruits:
    print(f"현재 과일: {fruit}")
```
> `fruits` 리스트의 각 요소가 `fruit` 변수에 차례로 할당되며 `for` 블록이 세 번 실행됩니다.

**예제 2: `range()` 함수와 함께 사용하기**
`range()` 함수는 특정 범위의 숫자를 생성해주어, 정해진 횟수만큼 반복할 때 매우 유용합니다.

- `range(stop)`: 0부터 `stop-1`까지의 숫자를 생성
- `range(start, stop)`: `start`부터 `stop-1`까지의 숫자를 생성
- `range(start, stop, step)`: `start`부터 `stop-1`까지 `step` 간격으로 숫자를 생성

```python
# 0부터 4까지 5번 반복
for i in range(5):
    print(f"{i}번째 반복입니다.")

# 1부터 10까지의 짝수만 출력
for i in range(2, 11, 2):
    print(i, end=" ") # 2 4 6 8 10
```

## `while` 문: 특정 조건이 만족되는 동안 반복하기

`while` 문은 주어진 **조건식이 참(`True`)인 동안 코드 블록을 계속해서 반복**합니다. 반복 횟수가 정해져 있지 않고, 특정 상태가 될 때까지 반복해야 할 때 유용합니다.

**기본 구조:**
```python
while 조건식:
    # 조건식이 참(True)인 동안 실행될 코드
    # (주의: 조건식이 언젠가는 거짓이 되도록 상태를 변경해야 함)
```

**예제:**
```python
count = 5
while count > 0:
    print(f"카운트 다운: {count}")
    count -= 1  # count 값을 1씩 감소시켜 언젠가 반복이 끝나도록 함

print("발사!")
```
> **무한 루프 주의!**: `while` 문 안에서 조건식의 상태를 변경하는 코드가 없으면 조건이 영원히 참으로 유지되어 프로그램이 멈추지 않는 **무한 루프(Infinite Loop)**에 빠질 수 있습니다. 의도적으로 무한 루프를 만들고 내부에서 `break`로 탈출하는 경우도 있지만, 항상 탈출 조건을 명확히 해야 합니다.

## 반복 제어: `break`와 `continue`

반복문이 실행되는 도중에 흐름을 제어해야 할 때가 있습니다. 이때 `break`와 `continue`를 사용합니다.

### `break`: 반복 즉시 중단하기

`break`는 반복문을 즉시 탈출합니다. `for`문이든 `while`문이든 `break`가 실행되는 순간, 반복은 완전히 종료됩니다.

**예제:**
```python
# 리스트에서 특정 값을 찾으면 반복 종료
numbers = [1, 5, 9, 12, 17, 21]
for num in numbers:
    if num > 10:
        print(f"10보다 큰 첫 번째 숫자 {num}을(를) 찾았습니다.")
        break  # 반복 중단
```

### `continue`: 현재 반복만 건너뛰기

`continue`는 현재 실행 중인 반복 스텝을 즉시 중단하고, **다음 반복 스텝으로 넘어갑니다.** 반복문 자체를 탈출하지는 않습니다.

**예제:**
```python
# 1부터 10까지의 숫자 중 홀수만 출력하기
for i in range(1, 11):
    if i % 2 == 0:  # 숫자가 짝수이면
        continue    # 아래의 print()를 실행하지 않고 다음 숫자로 넘어감
    print(i, end=" ") # 1 3 5 7 9
```

## 💡 반복문 꿀팁과 고급 활용법

### 1. `for-else` 문

`for` 문이 `break` 없이 **정상적으로 끝까지 실행되었을 때** 특정 코드를 실행하고 싶다면 `else` 블록을 사용할 수 있습니다.

**예제:**
```python
# 리스트에 특정 값이 있는지 검색하고, 없으면 메시지 출력
items = ["펜", "노트", "지우개"]
search_item = "연필"

for item in items:
    if item == search_item:
        print(f"'{search_item}'을(를) 찾았습니다.")
        break
else:
    # for 문이 break 없이 모두 실행된 후에만 이 블록이 실행됨
    print(f"'{search_item}'이(가) 리스트에 없습니다.")
```
> `while` 문도 `while-else` 구문을 동일하게 사용할 수 있습니다.

### 2. `enumerate()`: 인덱스와 값을 함께 얻기

리스트 등을 순회할 때 현재 요소의 인덱스 번호와 값을 동시에 얻고 싶다면 `enumerate()` 함수를 사용하면 매우 편리합니다.

**예제:**
```python
scores = [95, 88, 76, 100]
for index, score in enumerate(scores):
    print(f"{index + 1}번 학생의 점수는 {score}점입니다.")
```

### 3. `zip()`: 여러 시퀀스를 함께 묶어 순회하기

두 개 이상의 리스트(또는 다른 시퀀스)를 같은 인덱스끼리 짝지어 동시에 순회하고 싶을 때 `zip()` 함수를 사용합니다.

**예제:**
```python
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]

for name, age in zip(names, ages):
    print(f"{name}의 나이는 {age}세입니다.")
```
> **주의**: `zip`은 길이가 가장 짧은 시퀀스를 기준으로 반복이 종료됩니다.

### 4. 리스트 컴프리헨션 (List Comprehension)

`for` 문을 한 줄로 간결하게 작성하여 새로운 리스트를 만드는 강력하고 파이썬스러운(Pythonic) 방법입니다.

**기본 구조:**
`[표현식 for 항목 in 반복_가능_객체 if 조건]`

**예제:**
```python
# 1부터 10까지의 제곱수를 담은 리스트 만들기
squares = [x**2 for x in range(1, 11)]
print(squares)  # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

# 1부터 10까지의 짝수만 제곱한 리스트
even_squares = [x**2 for x in range(1, 11) if x % 2 == 0]
print(even_squares)  # [4, 16, 36, 64, 100]
```
> 리스트 컴프리헨션은 일반 `for`문보다 더 빠르고 가독성이 높아 파이썬 개발자들이 애용하는 기능입니다.

반복문은 프로그래밍에서 자동화를 구현하는 핵심 도구입니다. 각 반복문의 특징과 다양한 활용법을 익혀두면 복잡한 문제도 손쉽게 해결할 수 있습니다.
